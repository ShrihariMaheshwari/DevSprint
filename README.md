## ğŸš€ Getting Started

### Prerequisites
- Node.js 18+ (LTS recommended)
- npm 9+

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/devsprint.git

# Navigate to project directory
cd devsprint

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
